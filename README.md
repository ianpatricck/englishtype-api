# @ Englishtype API
## Englishtype Server-side

### Clone
```
$ git clone https://github.com/ianpatricck/englishtype-api
```

### Enter in project directory
```
$ cd englishtype-api
```

### Migrate and seed

Before that, configure the .env filesure env file.

```
$ yarn db migrate
$ yarn db seed
```

### Run the installation and startup commands
```
$ yarn install
$ yarn watch
$ yarn start
```
